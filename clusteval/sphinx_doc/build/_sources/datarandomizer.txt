Data Randomizer - Distorting Existing Data Sets
===============

ClustEval provides means to distort data sets. For a list of all available data randomizers head over `here <../../sphinx_doc_packages/build/de/clusteval/data/randomizer/package-index.html>`_.

Check :ref:`clusteval_extend_data_randomizers` for more information on how to extend ClustEval by new data randomizer.
