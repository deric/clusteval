ClusteringQualityMeasureValue¶

public class ClusteringQualityMeasureValue¶

This is a wrapper class for double values calculated by clustering quality measures.

This wrapper class allows for the fact, that iterations of parameter optimizations might not terminate. In this case toString() returns “NT”, where “NT” means “Not Terminated”. The factory method getForNotTerminated() returns such objects.

Author:Christian Wiwie

parseFromString¶

public static ClusteringQualityMeasureValue parseFromString(String stringValue)¶

This method returns a clustering quality measure value wrapper object corresponding to the given string.

If the string equals NT, a wrapper object for a not terminated iteration is returned by invoking getForNotTerminated().

Otherwise the string is parsed as a double value and the result of getForDouble(double) is returned.

Parameters:
  • stringValue – A string representation of the clustering quality.

A clustering quality value wrapper object corresponding to the given string.
