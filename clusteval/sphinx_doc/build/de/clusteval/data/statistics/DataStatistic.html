DataStatistic¶

public abstract class DataStatistic extends Statistic¶

A data statistic is a type of Statistic, which corresponds to properties of a data configuration (dataset and goldstandard).

Data statistics correspond to DataAnalysisRun in the class hierarchy.

Author:Christian Wiwie
