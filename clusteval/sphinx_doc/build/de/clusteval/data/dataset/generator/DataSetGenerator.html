generatesGoldStandard¶

public abstract boolean generatesGoldStandard()¶

If your dataset generator also creates a goldstandard for the generated dataset, this method has to return true.

If a goldstandard is to be created, it is going to be stored under the same folderName and with the same fileName as the dataset, but within the goldstandard directory of the repository.

Returns:A boolean indicating, whether your dataset generator also generates a corresponding goldstandard for the created dataset.
