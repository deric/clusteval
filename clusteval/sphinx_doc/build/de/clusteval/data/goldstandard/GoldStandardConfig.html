GoldStandardConfig¶

public class GoldStandardConfig extends DumpableRepositoryObject¶

A goldstandard configuration encapsulates options and settings for a goldstandard. During the execution of a run, when goldstandards are used, settings are required that control the behaviour of how the goldstandard has to be handled.

A goldstandard configuration corresponds to and is parsed from a file on the filesystem in the corresponding folder of the repository (see Repository.goldStandardConfigBasePath and GoldStandardConfigFinder).

There are several options, that can be specified in the goldstandard configuration file (see parseFromFile(File)).

Author:Christian Wiwie
