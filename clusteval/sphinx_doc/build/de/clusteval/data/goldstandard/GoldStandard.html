GoldStandard¶

public class GoldStandard extends RepositoryObject¶

A wrapper class for a goldstandard on the filesystem.

Author:Christian Wiwie

Fields¶

clustering¶

protected Clustering clustering¶

This attribute holds the clustering that corresponds to the goldstandard. Every goldstandard can be interpreted as a clustering: A partition of the data objects into several groups.

See also: {@linkClustering}

parseFromFile¶

public static GoldStandard parseFromFile(File absGoldStandardPath)¶

This method parses a goldstandard from a file. Since goldstandard files do not require a header, this method simply creates a wrapper object for the file on the filesystem.

The actual parsing of the clustering contained in the goldstandard file happens later in the loadIntoMemory() method.

Parameters:
  • absGoldStandardPath – The absolute path to the goldstandard file that should be parsed.
Throws:
  • NoRepositoryFoundException –
  • GoldStandardNotFoundException –
  • RegisterException –
Returns:

The goldstandard object.
