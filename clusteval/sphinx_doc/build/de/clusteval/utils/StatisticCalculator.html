StatisticCalculator¶

public abstract class StatisticCalculator<T extends Statistic> extends RepositoryObject¶

Together with every statistic class comes a calculator class, which is a factory class for the corresponding statistic. The calculator has a calculate() method, which calculates, stores and returns a statistic object.
