ProgramParameter¶

public abstract class ProgramParameter<T> extends RepositoryObject¶

An object of this class corresponds to a parameter of a program.

Therefore a program parameter has a reference to a program configuration in which it was defined.

A program parameter has a certain name unique for the program configuration, a description, a minimal value minValue, a maximal value maxValue and a default value def.

evaluateDefaultValue¶

public abstract T evaluateDefaultValue(DataConfig dataConfig, ProgramConfig programConfig)¶

This method evaluates the string representation of the default value def to a value corresponding to the dynamic type of this object, e.g. in case this parameter is a double parameter, it is evaluated to a double value.

The method requires a data and program configuration, since the string representation can contain a placeholder of a internal variable which is replaced by looking it up during runtime. $(meanSimilarity) for example is evaluated by looking into the data and calculating the mean similarity of the input.

Parameters:
  • dataConfig – The data configuration which might be needed to evaluate certain placeholder variables.
  • programConfig – The program configuration which might be needed to evaluate certain placeholder variables.
Throws:
  • InternalAttributeException –
Returns:

The evaluated value of the def variable.
