RProgram¶

public abstract class RProgram extends Program implements RLibraryInferior¶

A type of progam that encapsulates a program embedded in R.

Note that the name of the method in R is not itself written in plaintext in the exec(de.clusteval.data.DataConfig,de.clusteval.program.ProgramConfig,String[],java.util.Map,java.util.Map) method of RPrograms, but instead it is defined in the corresponding program configuration in the invocation format parameter.

Author:Christian Wiwie
