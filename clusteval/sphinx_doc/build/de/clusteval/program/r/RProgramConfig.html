RProgramConfig¶

public class RProgramConfig extends ProgramConfig¶

Objects of this class encapsulate the configuration of rprograms.

In principle this class does the same as a regular ProgramConfig, except that it takes the invocatin formats from the rprogram.

Author:Christian Wiwie

  • program – The program this program configuration belongs to.
  • outputFormat – The output format of the program.

  • params – The parameters of the program.
  • optimizableParameters – The parameters of the program, that can be optimized.
