RunSchedulerThread¶

public class RunSchedulerThread extends ClustevalThread¶

Threads of this class are responsible for scheduling, creating, starting and terminating runs.

The methods schedule(String,String), scheduleResume(String,String) and terminate(String,String) of this thread are invoked by the backend server ClustevalBackendServer which in turn gets its commands from a client.

Author:Christian Wiwie

schedule¶

public boolean schedule(String clientId, String runId)¶

This method is invoked by ClustevalBackendServer.performRun(String,String) and schedules a run. As soon as ressources are available this run is then performed in an asynchronous way in its own thread.

Parameters:
  • clientId – The id of the client that wants to schedule this run.
  • runId – The id of the run that should be scheduled
Returns:

True, if successful. That means, a run with the given id must exist and the run has not been scheduled already.
