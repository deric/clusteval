AnalysisRun¶

public abstract class AnalysisRun<S extends Statistic> extends Run¶

An abstract class for all run types, that involve conduction of any analyses. This can involve analyses of data and/or clustering results.

An analysis run has a list of statistics in statistics, that should be assessed for the objects of analysis.
