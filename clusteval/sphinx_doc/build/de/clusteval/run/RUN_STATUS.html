RUN_STATUS¶

public enum RUN_STATUS¶

During execution of a run, the corresponding object has a certain status.

Enum Constants¶

FINISHED¶

public static final RUN_STATUS FINISHED¶

When the job containing all its subprocesses is completely finished, its status is changed to FINISHED.

INACTIVE¶

public static final RUN_STATUS INACTIVE¶

When a job is just inactive

RUNNING¶

public static final RUN_STATUS RUNNING¶

As soon as ressourcess are available, the scheduler starts the job and its status is changed to RUNNING.

SCHEDULED¶

public static final RUN_STATUS SCHEDULED¶

Immediately after giving the command to execute the job, it is added to the queue of the de.clusteval.framework.threading.RunSchedulerThread with status SCHEDULED.

TERMINATED¶

public static final RUN_STATUS TERMINATED¶

When a job is terminated before it was finished, its status is changed to TERMINATED.
