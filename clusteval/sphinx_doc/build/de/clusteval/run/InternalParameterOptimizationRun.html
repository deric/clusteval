InternalParameterOptimizationRun¶

public class InternalParameterOptimizationRun extends ExecutionRun¶

A type of execution run that does the same as ParameterOptimizationRun, by using a programs internal parameter optimization mode instead of doing the parameter optimization itself within the framework.

Author:Christian Wiwie
