ClusteringRunRunnable¶

public class ClusteringRunRunnable extends ExecutionRunRunnable¶

A type of an execution runnable, that corresponds to ClusteringRun and is therefore responsible for performing only a single clustering.

In doRun() a ClusteringRunRunnable executes only a single iteration.

Author:Christian Wiwie

ClusteringRunRunnable¶

public ClusteringRunRunnable(RunSchedulerThread runScheduler, Run run, ProgramConfig programConfig, DataConfig dataConfig, String runIdentString, boolean isResume, Map<ProgramParameter<?>, String> runParams)¶
Parameters:
  • runScheduler – The run scheduler that the newly created runnable should be passed to and executed by.
  • run – The run this runnable belongs to.
  • runIdentString – The unique identification string of the run which is used to store the results in a unique folder to avoid overwriting.
  • programConfig – The program configuration encapsulating the program executed by this runnable.
  • dataConfig – The data configuration used by this runnable.
  • isResume – True, if this run is a resumption of a previous execution or a completely new execution.
