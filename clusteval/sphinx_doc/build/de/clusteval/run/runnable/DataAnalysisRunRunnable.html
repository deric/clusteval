DataAnalysisRunRunnable¶

public class DataAnalysisRunRunnable extends AnalysisRunRunnable<DataStatistic, DataAnalysisRunResult, DataAnalysisIterationWrapper, DataAnalysisIterationRunnable>¶

A type of analysis runnable, that corresponds to DataAnalysisRun and is responsible for analysing a data configuration (dataset and goldstandard).

Author:Christian Wiwie

DataAnalysisRunRunnable¶

public DataAnalysisRunRunnable(RunSchedulerThread runScheduler, Run run, String runIdentString, boolean isResume, DataConfig dataConfig, List<DataStatistic> statistics)¶
Parameters:
  • runScheduler – The run scheduler that the newly created runnable should be passed to and executed by.
  • run – The run this runnable belongs to.
  • runIdentString – The unique identification string of the run which is used to store the results in a unique folder to avoid overwriting.
  • dataConfig – The data configuration to be analysed by this runnable.
  • statistics – The statistics that should be assessed during execution of this runnable.
  • isResume – True, if this run is a resumption of a previous execution or a completely new execution.
