RunDataAnalysisRunRunnable¶

public class RunDataAnalysisRunRunnable extends AnalysisRunRunnable<RunDataStatistic, RunDataAnalysisRunResult, RunDataAnalysisIterationWrapper, RunDataAnalysisIterationRunnable>¶

A type of analysis runnable, that corresponds to RunDataAnalysisRun and is responsible for analysing a run result together with a data analysis result.

Author:Christian Wiwie

RunDataAnalysisRunRunnable¶

public RunDataAnalysisRunRunnable(RunSchedulerThread runScheduler, Run run, String runIdentString, boolean isResume, List<String> uniqueRunAnalysisRunIdentifier, List<String> uniqueDataAnalysisRunIdentifier, List<RunDataStatistic> statistics)¶
Parameters:
  • runScheduler – The run scheduler that the newly created runnable should be passed to and executed by.
  • run – The run this runnable belongs to.
  • runIdentString – The unique identification string of the run which is used to store the results in a unique folder to avoid overwriting.
  • uniqueRunAnalysisRunIdentifier – The identifiers of run analysis run results.
  • uniqueDataAnalysisRunIdentifier – The identifiers of data analysis run results.
  • statistics – The statistics that should be assessed during execution of this runnable.
  • isResume – True, if this run is a resumption of a previous execution or a completely new execution.
