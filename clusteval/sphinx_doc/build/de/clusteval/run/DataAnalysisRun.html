DataAnalysisRun¶

public class DataAnalysisRun extends AnalysisRun<DataStatistic>¶

A type of analysis run that conducts analyses of data inputs encapsulated by data configurations.

A data analysis run has a list of data configurations in dataConfigs , that should be assessed during execution of the run. Additionally they inherit a list of data statistics in AnalysisRun.statistics that should be assessed for every data configuration.

Author:Christian Wiwie
