RunDataAnalysisRun¶

public class RunDataAnalysisRun extends AnalysisRun<RunDataStatistic>¶

A type of analysis run that conducts analyses on both run results and data inputs together.

A run data analysis run has a list of unique run analysis run identifiers in uniqueRunAnalysisRunIdentifiers and a list of unique data analysis run identifiers in uniqueDataAnalysisRunIdentifiers, that should be assessed during execution of the run. Additionally they inherit a list of run statistics in AnalysisRun.statistics that should be assessed for every pair of run analysis and data analysis run identifier.

Author:Christian Wiwie
